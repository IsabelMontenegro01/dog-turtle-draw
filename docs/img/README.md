# Imagens - Turtle Draw

## turtle_result.png

Screenshot do resultado final do turtle desenhando os contornos extraídos.

**Como adicionar:**
1. Tire uma screenshot do turtlesim desenhando
2. Salve em PNG com o nome `turtle_result.png` nesta pasta
3. Será automaticamente exibida em `documentacao.md`

**Requisitos:**
- Resolução: mínimo 400x400px
- Formato: PNG ou JPG
- Conteúdo: Resultado visual do turtle desenhando contornos
